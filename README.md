# check-name-app

## How to run web

### Prepare env
Install node.js
- go to this link https://nodejs.org/en/
- download version 8.9.4 LTS
- run node-v8.9.4-x64.msi

see installing step in http://blog.teamtreehouse.com/install-node-js-npm-windows

Install vscode
- download and install https://code.visualstudio.com/download

### Run web
- open vscode
- on menu, select file -> open folder
- select project folder
- type ctrl + \` for open terminal
- type `npm install` in terminal for install package
- type `npm run build` in terminal for build web
- type `npm run start` in terminal for run web
- open http://localhost:3000
